# KREIT - Universal Property Intelligence Platform

A comprehensive real estate intelligence platform with deterministic formula-based scoring using live data from 80+ verified FREE APIs across 200+ countries.

## Features
- **Global Property Discovery** - Search properties worldwide
- **KREIT Score** - Transparent, trustworthy scoring (0-100)
- **Database Explorer** - AI-powered data exploration
- **Personalized Analysis** - Tailored to your investment profile

## Technology Stack
- Frontend: React, TypeScript, Tailwind CSS, Shadcn UI
- Backend: Express.js, PostgreSQL, Drizzle ORM
- AI: OpenAI GPT-4o-mini
- APIs: 80+ verified data sources
